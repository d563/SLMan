httpd
=====
::

    yum -y install httpd

::

    chkconfig httpd on
    /etc/init.d/httpd start

.. index:: /etc/sysconfig/iptables

::

    sed -i.org \
    -e '/--dport 22/a\
    -A INPUT -p tcp -m state --state NEW -m tcp --dport 80 -j ACCEPT' \
    /etc/sysconfig/iptables

    /etc/init.d/iptables restart
