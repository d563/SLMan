github
------
::

    ssh -T git@github.com

::

    cd /var/www/html
    git init .
    touch .nojekyll
    git add .
    git commit -mstart
    git remote add origin git@github.com:d563/SLMan.git
    git push -u origin master:gh-pages

    git clone git@github.com:d563/SLMan.git /var/www/html
