ローカル環境¶

基本方針¶

  • Windows を使ってはいけない職場は少ないので素直に Windows に慣れる
  • cmd.exe は使いにくいので Mintty から bash や管理用のコマンドが使える状態にしておく
  • 本気でサーバの挙動を調査するなら EC2 や VirtualBox で建てれば良い

Git Bash¶

https://gitforwindows.org/ からインストーラをダウンロードし、デフォルトの設定でインストールする。 （デフォルトで Mintty での起動が選ばれるはず）

.bash_profile はログインする（と言うより Mintty を起動する）たび、 .bashrc は bash コマンドを実行するたびに読み込まれるので、 .bash_profile から .bashrc を読み込むような記述にしておくと困らない。 （こららのファイルは c:\Users 配下のログインユーザ名のディレクトリに置く）

  • .bash_profile
    [[ -f ~/.bashrc ]] && . ~/.bashrc
  • .bashrc
    alias ls='ls --color=auto --show-control-chars'
alias ll='ls -l'
PS1='$(pwd) $ '
LANG=ja_JP.UTF-8
